# 文生图界面优化实现方案

## 一、需求分析

### 1. 风格示例展示优化
- 在创作中心下方添加风格示例卡片
- 支持11种风格，每种风格对应一张示例图片和提示词
- 响应式设计，适配不同屏幕尺寸

### 2. 案例演示帖子区域
- 卡片式布局展示案例
- 支持复制配置、查看详情、收藏功能
- 分类浏览和排序功能

## 二、技术方案

### 1. 风格示例展示功能

#### 1.1 新增文件
- `src/components/StyleExampleCard.vue`：风格示例卡片组件
- `src/assets/style-examples/`：存放11种风格的示例图片

#### 1.2 修改现有文件
- `src/views/HomeView.vue`：在创作中心下方添加风格示例展示区域
- `src/views/PromptGenerateView.vue`：添加风格示例展示

#### 1.3 数据结构设计
```typescript
interface StyleExample {
  id: string;           // 风格ID
  name: string;         // 风格名称
  imageUrl: string;     // 示例图片URL
  prompt: string;       // 示例提示词
}
```

### 2. 案例演示帖子区域

#### 2.1 新增文件
- `src/components/CaseCard.vue`：案例演示卡片组件
- `src/components/CaseFilter.vue`：案例分类筛选组件
- `src/components/CaseDetailModal.vue`：案例详情弹窗
- `src/composables/useCaseData.ts`：案例数据管理
- `src/types/case.ts`：案例相关类型定义

#### 2.2 修改现有文件
- `src/views/HomeView.vue`：添加案例演示区域
- `src/router/index.ts`：添加案例详情路由（可选）

#### 2.3 数据结构设计
```typescript
interface CaseItem {
  id: string;                 // 案例ID
  title: string;              // 案例标题
  description: string;        // 案例描述
  category: string;           // 案例分类
  style: string;              // 使用的风格
  imageUrl: string;           // 案例图片
  config: {
    prompt: string;           // 生成提示词
    styleId: string;          // 风格ID
    otherParams: any;         // 其他配置参数
  };
  likes: number;              // 点赞数
  views: number;              // 浏览量
  createdAt: string;          // 创建时间
  isFavorite: boolean;        // 是否收藏
}
```

## 三、实现步骤

### 1. 风格示例展示实现

#### 步骤1：准备风格示例图片
- 在`src/assets/style-examples/`目录下添加11种风格的示例图片，命名规则：`{styleId}.png`
- 例如：`xiaohongshu.png`、`poster_2k.png`等

#### 步骤2：创建StyleExampleCard组件
- 设计卡片布局，包含风格名称、示例图片和提示词
- 添加悬停效果和响应式设计

#### 步骤3：在HomeView.vue中集成
- 添加风格示例展示区域
- 从stylePrompts.ts获取所有风格配置
- 动态渲染风格示例卡片

### 2. 案例演示区域实现

#### 步骤1：创建案例数据管理
- 在useCaseData.ts中实现案例数据的获取、分类、排序等功能
- 支持本地存储收藏状态

#### 步骤2：创建CaseCard组件
- 设计案例卡片，包含标题、描述、分类、图片等
- 添加复制配置、查看详情、收藏按钮

#### 步骤3：创建CaseFilter组件
- 实现案例分类筛选功能
- 添加排序选项（最新、最热、推荐）

#### 步骤4：创建CaseDetailModal组件
- 展示案例完整配置信息
- 支持复制配置到剪贴板

#### 步骤5：在HomeView.vue中集成
- 添加案例演示区域
- 集成CaseCard和CaseFilter组件
- 实现案例分类浏览和排序功能

## 四、资源准备

### 1. 风格示例图片
- 需要准备11种风格的示例图片，建议分辨率：2048x2730（3:4）
- 图片应能清晰体现对应风格的特点

### 2. 示例提示词
- 为每种风格准备匹配的示例提示词，重点体现该风格的特点和色系
- 提示词应简洁明了，突出风格核心要素

### 3. 案例数据
- 准备10-20个演示案例，覆盖不同风格和场景
- 每个案例包含完整的配置信息

## 五、预期效果

### 1. 风格示例展示
- 用户选择风格后，可直观查看该风格的示例效果
- 示例提示词帮助用户理解风格特点
- 响应式设计，在不同设备上都有良好的展示效果

### 2. 案例演示区域
- 用户可浏览各种案例，了解不同风格的实际应用
- 支持快速复制案例配置，提高创作效率
- 分类和排序功能帮助用户快速找到感兴趣的案例
- 收藏功能方便用户保存常用案例

## 六、实施计划

1. **第一阶段**：实现风格示例展示功能
   - 创建StyleExampleCard组件
   - 准备风格示例图片和提示词
   - 集成到HomeView.vue

2. **第二阶段**：实现案例演示区域
   - 创建CaseCard、CaseFilter、CaseDetailModal组件
   - 实现案例数据管理
   - 集成到HomeView.vue

3. **第三阶段**：测试和优化
   - 测试所有功能的完整性
   - 优化响应式设计
   - 提升用户体验

通过以上实现方案，将有效提升文生图功能的用户体验，帮助用户更好地理解和选择风格，同时提供丰富的案例参考，提高创作效率。
